# DDoS & WAF Interview Prep 🎯

A structured collection of interview questions and answers for NOC Engineer, Application Security, and DDoS Protection roles.

Written from a practical NOC perspective — not textbook definitions. Each answer reflects how these concepts work in a real operations environment.


📁 DDoS Concepts


1. ddos-fundamentals-qa.md — What is DDoS, attack types, OSI layers, volumetric vs protocol vs application
2. ddos-attack-types-qa.md — SYN flood, UDP flood, ICMP flood, ACK flood, HTTP flood — deep dive Q&A
3. ddos-mitigation-qa.md — Null routing, rate limiting, scrubbing, BGP blackhole, RTBH
4. ddos-tools-qa.md — Grafana, Kentik, Pingdom — how and why you use them



📁 WAF & Application Security


1. waf-fundamentals-qa.md — What is a WAF, how it works, modes, rule sets
2. cwaf-operations-qa.md — Cloud WAF incident handling, false positives, rule tuning
3. owasp-top10-qa.md — OWASP Top 10 explained from a WAF/NOC perspective



📁 NOC Operations


1. noc-scenario-questions.md — "Walk me through how you'd handle..." scenario questions
2. noc-tools-process-qa.md — Jira, Confluence, escalation, SLA, on-call process Q&A
3. networking-fundamentals-qa.md — TCP/UDP, OSI model, BGP basics, SNMP, NetFlow — interview level



📁 Behavioral & HR


1. behavioral-questions.md — Tell me about yourself, strengths, handling pressure, team situations
2. career-gap-answer.md — How to address a career gap confidently



🧠 How to Use This


1. Read through each file before interviews
2. Don't memorise — understand the concept and answer in your own words
3. Use the scenario questions to practise speaking out loud
4. Add your own notes and experiences to each answer
